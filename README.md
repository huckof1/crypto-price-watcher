# Crypto Price Watcher

Простой Python-скрипт, который получает текущую цену Bitcoin и Ethereum через публичный API Binance и выводит их в консоль.

## Как запустить
1. Установить зависимости:
   pip install -r requirements.txt
2. Запустить:
   python main.py

## Возможности
- Получение текущей цены BTC и ETH
- Минимальная зависимость (requests)
